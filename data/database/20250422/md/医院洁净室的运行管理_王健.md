# 医院洁净室的运行管理  

厦门港湾咨询监理有限公司王健\*  

摘要医院洁净室是一个多专业的系统工程，科学有效的管理，有益于体现它的投资效益和社会效益。关键词医院洁净室运行管理节能降耗  

# 0引言  

医院洁净室由于它涉及到净化空调、医用气体、给排水、强弱电、智能化、室内装修、消防以及设备安装等多个分项工程，加之初期投资大，运行成本高，可以说：医院洁净室运行管理科学合理与否，直接反映了一个现代化医院的医疗管理水平和后勤保障水平。同时，洁净室设备是医院耗能大户，如何科学合理地运行管理主要设备，节能降耗，是现阶段面临的主要问题。

医院洁净室不仅指洁净手术部，也包括有洁净度要求的血液病房,ICU和隔离病房等。  

# 1健全规章制度  

# 1.1健全医疗使用制度  

健全感染监控领导小组，明确各岗位责任和职能，根据医院的实际情况和需要，细化落实到人，严格人流，物流界限和管理，从细节和微观入手，强化内部卫生，严格遵守和执行净化程序。  

定期进行细菌检测和记录，明确各项指标的最值和预警标准，对异常情况会判断，会处理，能汇报。落实交接班制度，人员培训制度，爱护建筑饰面和配置设备，规范使用，避免野蛮的非正常操作。规范物品摆放。  

# 1.2健全设备管理制度  

动态运转设备，如电动机、风机等，参考机械设备运行管理制度，完善设备巡检制度，交接班制度等规章制度。  

对于洁净室的维保，需严格执行《手术部医院感染预防与控制技术规范》要求，定期检测洁净室的风速、压差、风量以及湿度等指标，做好记录和数据归档，保留第一手资料。视情况，一定周期（建议每半年）委托有检测资质的单位或部门，对洁净室的洁净度、噪声、照度、压差、风速或换气次数做一次检测，根据检验数据和结果，确定是否更换过滤器或更换过滤器的种类和数量。  

# 2.选择专业维保公司  

日常维保的质量，直接影响洁净室的使用效果和使用寿命，对医疗工作的正常、有序的开展至关重要；现阶段，靠医院内部人员管理，在专业技能和人员配备上，很难满足使用要求。因此，选择专业的维护保养公司，势在必行。但由于洁净室的维保，涉及专业广，技术含量高，无固定取费标准，目前仍处于摸索阶段，没有固定参考模式。  

# 3确保设备正常运转，降低设备运行成本  

# 3.1确保设备正常运转  

洁净室空调不能等同于一般中央空调，需要有专人负责。每天不固定时间巡检，定期分析所记录的数据。尤其对空气处理机组的进出口风量，压差，洁净室的温度，湿度等主要指标和影响该指标的因素，要掌握规律后加以监控，做到未雨绸缪，不打无准备之仗。  

![](output\images\医院洁净室的运行管理_王健/d9e8780ddf3c83547997b24a748fdcb668ad267634f055ae13d8aa88dfdf791f.jpg)  

对洁净室的回风过滤器，新风的初效过滤器，机组内的中效和中高效或亚高效过滤器，洁净室送风高效过滤器要定期更换，清洁，消毒。由于厦门地区空气质量优良，一般地，初、中效过滤器每月检查清洗，检查有无泄漏或堵塞，发现后要及时更换，高效过滤器每半年配合检测确定更换与否：对于高效过滤器参考指标，除室内洁净度要求外，还应参考高效过滤器终阻力值，目前国内高效过滤器初阻力一般约 $120\mathrm{Pa}$ （中国建筑科学研究院空调所专利产品初阻力可达到 $\mathsf{i}20\mathsf{P a}$ ），终阻力约 $350{\mathrm{-}}400{\mathrm{Pa}}$ 。空调箱建议定期熏蒸，效果更明显。  

# 3.2降低设备运行成本  

降低设备运行成本，主要体现在电能的消耗，辅材的更换和设备维修费用上，合理维护保养设备，不但可以降低上述费用，更能延长机组寿命，保障使用效果。南方地区空气湿度大，本身新风中湿度超标，直接造成空气处理机组中湿度经常超过 $80\%$ ，而过高的湿度，容易产生细菌的滋生和繁殖，不利于保证净化效果。同时，一般设计工况都参考国标规定：洁净室湿度控制在 $40\%-60\%$ ，所以，经常要采用电除湿技术，而此技术电能消耗严重，少则十几千瓦，多则几十千瓦，效果还不是很理想。另外，系统运行一段时间后，集中在初、中、高效过滤器上的风阻，也会造成电能的浪费和损耗，降低设备使用效率，增加设备运行成本。  

# 4结语  

目前，国内很大一部分医院的设备实际运行工况与设计工况相差甚远，甚至大相径庭，虽然他们的设计理念和建造工艺比较先进，但缺乏严格有效的机制去管理和使用，还处于粗放型管理阶段，所以很难发挥出应有的效果，造成能源的浪费和不要的流失，因此，节能降耗，严禁设备关键在于加强设备运行管理。
